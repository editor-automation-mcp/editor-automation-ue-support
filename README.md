# Editor Automation — Support

Public support tracker for the **Editor Automation RPC Gateway** Unreal Engine plugin.

This repository contains **no plugin source**. It exists only to collect and triage support, and it is the **only** support channel — reports are not handled through the private release repository.

## Where to report

| What | Where |
|---|---|
| A bug, or a critical gap in an existing feature | **[Open an Issue](https://github.com/editor-automation-mcp/editor-automation-ue-support/issues/new?template=bug_report.yml)** |
| A new or extra feature idea | **[Discussions → Ideas](https://github.com/editor-automation-mcp/editor-automation-ue-support/discussions/new?category=ideas)** (votable) |
| An install or usage question | **[Discussions → Q&A](https://github.com/editor-automation-mcp/editor-automation-ue-support/discussions/categories/q-a)** |

## Filing a good bug report

Include the plugin version (from the `.uplugin` `VersionName`), your Unreal Engine version and OS, a **minimal reproduction**, and what you expected versus what happened. Keep root-cause speculation separate from what you actually observed.

If you drive the plugin with an AI coding agent through its MCP, the agent can draft a report and open a prefilled draft for you — it will never submit on its own; you always review and submit.
