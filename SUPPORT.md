# Support

- **New / extra feature idea** → [Discussions → Ideas](https://github.com/editor-automation-mcp/editor-automation-ue-support/discussions/new?category=ideas)
- **Install / usage question** → [Discussions → Q&A](https://github.com/editor-automation-mcp/editor-automation-ue-support/discussions/categories/q-a)

This repository is the only support channel. The plugin source is not here.
